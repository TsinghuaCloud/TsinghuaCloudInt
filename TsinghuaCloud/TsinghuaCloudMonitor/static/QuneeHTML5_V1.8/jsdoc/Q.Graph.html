Class: Graph

Q. Graph

new Graph(div)

图形组件类，可以添加各种图形元素（节点，连线，分组等），以及图形元素上挂载图形，实现丰富的图形展示效果，可以实现拓扑图，组织图，流程图，机房图等
Parameters:
Name Type Description
div HTMLElement | String 图形组件所在的HTML节点，或者HTML元素的id名称
